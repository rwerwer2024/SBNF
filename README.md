This repository contains the source code for the SBNF image fusion model.

Usage
1. Training
To train the model, execute the train.py script:

bash
python train.py

2. Inference
To perform inference and generate fused images, execute the test.py script:

bash
python test.py

3. Evaluation
To calculate objective evaluation metrics, execute the metric_test.py script:

bash
python metric_test.py
